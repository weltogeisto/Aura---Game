# AURA OF THE 21ST CENTURY

## Subtitle: "One Shot"

*A satirical destruction game exploring value, art, and absurdity through precision ballistics.*

---

## Core Concept

The player has **one pistol-caliber shot** to inflict maximum "cultural damage" in iconic interiors. The game interrogates what we value and why—blending dark humor, Dadaist philosophy, and challenging ballistics puzzles.

## Technical Architecture

### The Value Mesh

```
┌────────────────────────────────────────────────┐
│  360° HDR Cube Map (three-wall panoramic view) │
├────────────────────────────────────────────────┤
│  16-bit Grayscale Value Texture               │
│  - Hex-mapped: #FFFFFF = max value            │
│  - Variable density per object type           │
│  - Material properties embedded               │
├────────────────────────────────────────────────┤
│  Destruction Shader Layer                      │
│  - Radial damage propagation from impact      │
│  - Material-specific spread (canvas tears     │
│    wider than bronze dents)                   │
├────────────────────────────────────────────────┤
│  Systemic Trigger Network                      │
│  - Infrastructure nodes (sprinklers, HVAC)    │
│  - "Zone Wipe" chain reactions                │
└────────────────────────────────────────────────┘
```

### Value Grid Logic

| Element Type | Grid Density | Value Distribution |
|--------------|--------------|-------------------|
| Masterpiece painting | 20×30 micro-cells | Equal per cell (value = intactness) |
| Sculpture | 3D mapped cells | Higher at faces, hands, details |
| Architecture | Low density | Minimal unless historically significant |
| Decorative arts | Medium density | Concentrated at craftsmanship points |
| Easter eggs | Single hidden cell | Absurdly high or negative values |

### Ballistics Engine

| Variable | Implementation |
|----------|----------------|
| Gravity | Realistic 9mm/.45 ACP drop |
| Wind | Indoor drafts, HVAC currents |
| Aim stability | "Breath Sway" (pulse jitter) — crosshair aligns in millisecond windows |
| Obstacles | Ricochet physics + glass refraction index |
| Penetration | Material-specific (canvas, wood, bronze, glass) |

## Post-Shot Mechanics

### "The Appraisal" Phase
- Sotheby's hammer falls
- Receipt prints itemized damage
- Example: "Oil Paint: €50 → Cultural Significance: €800,000,000"

### "The Critic's Review"
- Snobby art critic voiceover evaluates your shot
- Unique lines per target and scenario

### "The Insurance Adjuster"
- AI appraiser argues your damage estimate
- Players can contest, triggering absurdist debates

### "Provenance Roulette"
- Random modifier: some items revealed as forgeries (value → €0)
- Others discovered as "lost masterpieces" (multiplier)

### "The Curator's Revenge"
- Rare event: curator throws body in front of artwork
- Moral dilemma: "Human Value vs. Art Value"

## Easter Egg Philosophy

Each scenario contains TWO hidden easter eggs:

1. **Dadaist Easter Egg** — Absurdist, philosophical, critiques value itself
2. **Systemic Easter Egg** — Chain-reaction, infrastructure collapse, maximum damage

Players discover one randomly per playthrough, encouraging replay.

## Visual/Audio Design Notes

**Aesthetic:** Hyperreal photography with surrealist UI elements (Magritte-style floating frames, Dalí-melting scoreboards)

**Sound:**
- Shot: Realistic, echoing in marble/wood spaces
- Impact: Material-specific (canvas tear, glass shatter, bronze ping)
- Score reveal: Ironic classical music (Strauss waltzes, papal choir samples)

**Post-Shot:**
Slow-motion bullet cam showing trajectory, impact, and "damage bloom" spreading across the value grid as numbers cascade.
